# AI and Machine Learning-Based Skin Cancer Classification

## Overview
The NHS receives approximately 1.2 million annual referrals for skin cancer, with only 659 dermatologists in England. 60% are considered urgent, but only 6% classified as malignant, which causes unnecessary strain on the NHS. This project aimed to reduce the number of unnecessary referrals by developing a mobile application, intended as a tool for general practitioners (GPs), to classify benign and malignant skin lesions. Existing models lack transparency in results and bias towards lighter skin tones, decreasing user trust and raising issues with training datasets.
